Offical way to create react by react-app but vite provide fast development exp;

1 first create pkg write in terminal (npm i gh-pages)

2 vite.config.js in define-confing write (base: "/repo-name/");

3 This step is not imp but best practice write ("homepage:" "repo-link") in pacakage.json

4 write in pakage.json in "scripts" ("deploy": "gh-pages -d dist")

dist is the build and build is production ready code we reploy dist or we say build not react app cuz browser didnt know react

5 Then push code in repo

6 write in terminal npm run build (dist will be create in folder)

7 npm run deploy

in git-hub repo new branch will be created and dist will be deployed in new-branch name (gh-pages) branch;
